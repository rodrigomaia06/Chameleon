Chameleon
====
A python program to encrypt and decrypt with the chameleon cipher.

## Prerequisites:
* Having python3 installed on your machine.

## Usage:
1step - Clone the repository or download it.

2step - You need to have your original deck and display it on the file "deck.txt". The first line of the file is the card on the top of the deck, and the last line of the file is the last card on the deck. The folder has an example of the "deck.txt" and you need to follow the structure for each line "(value of the card)_(suits)-(color of the card)".

3step - Run the program and choose the desired option.
